# חבילת מערכת בקרת איכות - נתיבי ישראל

מה עושים:
1. מחליפים/מעתיקים את תיקיית `app` לפרויקט Next.js שלך.
2. מריצים בטרמינל: `npm install` ואז `npm run dev`.
3. אם עובדים עם Supabase: מריצים לפי הסדר את הקבצים בתיקיית `supabase`.
4. להעלאה ל-Vercel: מעלים את הפרויקט ל-GitHub ואז Deploy ב-Vercel.

החבילה כוללת מסך תהליכי בקרה, חיבור לרשימות תיוג/RFI/אי-התאמות, ריכוזים, תעודות מעבדה וקבצי SQL.
